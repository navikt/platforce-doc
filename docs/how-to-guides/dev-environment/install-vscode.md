# Install VS Code

How to install Visual Studio Code on your computer.

## Prerequisites

=== "Windows"

    - [Scoop](install-scoop.md)

=== "macOS"

    - [Homebrew](install-homebrew.md)

## Installation

=== "Windows"

    ```PowerShell
    scoop bucket add extras
    scoop install vscode
    ```

=== "macOS"

    ```bash
    brew install --cask visual-studio-code
    ```
