# Install Flxbl SFP

How to install Flxbl SFP on your computer.

## Prerequisites

=== "Windows"

    - [Salesforce CLI](install-salesforce-cli.md)

=== "macOS"

    - [Salesforce CLI](install-salesforce-cli.md)

## Installation

=== "Windows"

    ```PowerShell
    npm install -global @flxbl-io/sfp
    ```

=== "macOS"

    ```bash
    npm install -global @flxbl-io/sfp
    ```
