# Install Git

How to install Git on your computer.

## Prerequisites

=== "Windows"

    - [Scoop](install-scoop.md)

=== "macOS"

    - [Homebrew](install-homebrew.md)

## Installation

=== "Windows"

    ```PowerShell
    scoop install git
    ```

=== "macOS"

    ```bash
    brew install git
    ```
