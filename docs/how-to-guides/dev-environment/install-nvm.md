# Install NVM

How to install Node Version Manager on your macOS computer.

## Prerequisites

=== "Windows"

    - [Scoop](install-scoop.md)

=== "macOS"

    No prerequisite needed.

## Installation

=== "Windows"

    `scoop install main/nvm`

=== "macOS"

    ```bash
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh
    ```
